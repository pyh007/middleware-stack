# MySQL 学习路线与进度

## 掌握等级

| 等级 | 完成标准 |
|---|---|
| L0 未学习 | 不能稳定解释概念 |
| L1 能解释 | 能用自己的话回答核心问题 |
| L2 能验证 | 能独立运行实验并解释证据 |
| L3 能诊断 | 能从执行计划或监控中定位问题 |
| L4 能决策 | 能结合业务、数据量和风险说明取舍 |

不要把“看完”记录为“掌握”。一个主题至少达到 L2，才进入下一个主题。

## 每个模块的验收清单

- [ ] 三分钟内讲清楚核心概念、机制和适用条件。
- [ ] 不看答案独立跑通实验。
- [ ] 能预测关键 SQL 的执行计划或锁行为。
- [ ] 能制造一个反例并说明为什么失败或变慢。
- [ ] 能回答对应面试题，并说明至少一个副作用。
- [ ] 能写出生产上线前的验证与回滚方法。
