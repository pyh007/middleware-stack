# 主动回忆与间隔复习

复习必须先回忆再重读。完成一个模块后约在第 1、3、7、14、30、90 天复习，并把日期、掌握等级和实验数值写入 [ROADMAP](../ROADMAP.md)。

```bash
make -C redis review-list
make -C redis review
uv run --project . python redis/study/scripts/review.py --topic 并发一致性 --count 2 --seed 42
```

一次 15 分钟复习分为四步：3 分钟看 [速查图](../CHEATSHEET.md)，4 分钟闭卷回答三题，5 分钟重跑关联实验，3 分钟口述“结论 → 机制 → 证据 → 边界 → 副作用 → 生产验证”。答错时先记录错因，再运行题目关联路径；不要先看答案获得熟悉感。

评分建议：0 分为无从回答，1 分为只会术语，2 分为能解释机制，3 分为能引用实验证据，4 分为能给生产边界和回滚。连续两轮低于 3 分则降一级并缩短下一次间隔；达到 4 分且能修改变量重跑才延长间隔。
